Registering Users POST

For registering users you can use below endpoint and you can provide:

  • name
  • email
  • password

/register

Logging Users in POST

For logging in you can use below endpoint and get a BEARER TOKEN to modify data, you can provide:

  • email
  • password

/login

Logout Users POST

You can use this endpoint to terminate users' sessions.

/logout

Fetching All Links GET

To retrieve all the authenticated users' links. Use endpoint below with provided information.

/profile

A Single Link GET

To get relevant information about a shortened link.

/{slug}

Adding a New Link POST

To add a new link to database you can use below endpoint with.

  • target

/profile/add

Updating a Link PUT

To update a link that belongs to you use below endpoint.

  • slug
  • target

/profile/edit

Deleting a link DELETE

To delete a link that belongs to you use below endpoint.

  • slug
